---# 스프링을 만지자(이상한뜻 아님) - 1일차

스프링에 대한 이해도와 그런것들이 부족하다고 느껴져서
스프링부트와 mysql과 리엑트(타입스크립트)를 이용한 게시판을 만들며 나의 능력을 한단계 올릴려고한다.

# 1. 쌩으로 만들게?
당연히 아니다.
이번 공부에서 내가 참고한 영상은
https://youtube.com/playlist?list=PLbq5jHjpmq7q-Td2jOXtpf7SD5c53RqXh&si=BKmv2YQKRxRA2QQb
서타몽의 영상들이다. 굉장히 좋으니 한번 보도록 하자.
근데 개념들은 알아서 찾도록
<br><br><br>
# 2. 첫번째 이야기
나는 우선 내가 만들 사이트의 간단한 인풋과 아웃풋을 짰다.
![](https://velog.velcdn.com/images/pobi/post/36ac0633-1b69-461b-b2ae-1172240fde57/image.png)
이런식으로 무엇을 요청하고 무엇을 받는지와 또 에러가났을때는 무엇을 줘야하는지 이런식으로 짜서 interface.txt에 넣어놓고 만들면서 보고있다.
<br><br><br>
# 3. 데이터베이스를 만들자
내가 만들 서비스는 정말 간단한 게시판 서비스이기 때문에
데이터베이스를 만드는데에는 어렵지는 않았다.
간단한 데이터베이스 그림을 그렸고 그것을 코드로 옮겼다.
정말 간단한 서비스이기 때문에 별거는 없다.
하지만 코드를 만들고 실행시키고 하는과정에서 중간에 유저가 만들어지지않는 오류가 있었지만 쿼리문 오류였고 지우고 다시 만드는 과정에서 시간이 좀 걸렸다.
![](https://velog.velcdn.com/images/pobi/post/fad3570a-7be3-4ebc-9979-7a8111146178/image.png)
이만큼 쿼리문을 쓴건 처음이라 무슨 에러가뜰까 조마조마 했지만
다행히 아무일도 일어나지않았다.


<br><br><br>
# 4. 의존성을 만들자
![](https://velog.velcdn.com/images/pobi/post/ba1ef758-8347-4eec-a9f4-39ff67788b20/image.png)
내가 사용한것은
Spring Web
validation
Spring Security
Spring Data JPA
MySQL Driver
Lombok을 사용하였다.
여기서 다른건 다 알겠는데 롬복을 처음본사람도 있을것이다.
그게 나다.

https://velog.io/@swjy1216/Lombok-%EB%A1%AC%EB%B3%B5%EC%9D%B4%EB%9E%80
롬복은 이분 설명이 짱인듯하니 보도록하자.
난 @Getter @Setter도 롬복님의 은혜인줄은 몰랐다.
쓰니까 인생이 달라지더라
장난이고 롬복이란 간단하게 그냥 @<-이거 가지고 반복되는 코드를 확줄이게 도와준다고 생각하면된다.
