---# 빼빼로 데이 특집: 나만의 디지털 빼빼로 만들기

## 들어가며
11월 11일, 빼빼로 데이가 다가오고 있습니다. 올해는 조금 특별하게 직접 빼빼로를 디자인하고 친구들과 공유할 수 있는 웹 서비스를 만들어봤습니다.

## 💡 Pepero Factory: 디지털 초콜릿 공장에서 당신만의 특별한 빼빼로를 만들어보세요!

🔗 배포 링크: https://51e8f075.pepero-factory.pages.dev
📦 GitHub 저장소: https://github.com/heodongun/pepero-factory
## 프로젝트 소개
Pepero Factory는 사용자가 웹에서 직접 빼빼로를 커스터마이징하고, 만든 작품을 다른 사람들과 공유할 수 있는 인터랙티브 웹 애플리케이션입니다.
주요 기능

빼빼로 커스터마이징: 다양한 토핑, 초콜릿 종류, 데코레이션 요소를 선택해 나만의 빼빼로 디자인
공유 기능: 완성한 빼빼로를 링크로 공유하여 친구들에게 선물
반응형 디자인: 모바일부터 데스크톱까지 모든 기기에서 원활한 경험
실시간 프리뷰: 선택한 옵션이 즉시 화면에 반영

## 기술 스택
프로젝트는 다음과 같은 기술들로 구성되어 있습니다:

Frontend: next
Deployment: Cloudflare Pages
Version Control: GitHub

## 개발 과정에서의 고민
#### 1. 사용자 경험 (UX)
빼빼로를 만드는 과정이 직관적이고 재미있어야 했습니다. 복잡한 인터페이스보다는 간단하면서도 창의성을 발휘할 수 있는 구조를 고민했습니다.
#### 2. 공유 기능 구현
각 사용자가 만든 빼빼로를 고유한 링크로 공유할 수 있도록 하는 것이 핵심 기능이었습니다. URL 파라미터를 활용한 상태 공유 방식을 채택했습니다.
#### 3. 성능 최적화
빼빼로 이미지와 다양한 데코레이션 요소들을 효율적으로 렌더링하는 것이 중요했습니다. 특히 모바일 환경에서도 부드러운 경험을 제공하기 위해 최적화에 신경 썼습니다.
#### 배포 및 운영
Cloudflare Pages를 통해 배포하여 빠른 로딩 속도와 안정적인 서비스를 제공하고 있습니다. GitHub와 연동되어 있어 main 브랜치에 푸시하면 자동으로 배포됩니다.
#### 마치며
빼빼로 데이를 맞아 시작한 작은 프로젝트였지만, 많은 것을 배울 수 있었던 경험이었습니다. 특히 사용자가 직접 콘텐츠를 만들고 공유하는 인터랙티브한 웹 경험을 구현하는 과정이 흥미로웠습니다.
여러분도 Pepero Factory에 방문해서 나만의 특별한 빼빼로를 만들어보세요! 그리고 만든 빼빼로를 소중한 사람들과 공유해보시는 건 어떨까요?
