---# 현실을 모방하던 것들이  언젠가 현실을 대신하기 시작하면

![](https://velog.velcdn.com/images/pobi/post/c6517d19-1255-43f0-956a-0e23018dd91d/image.png)

나는 가끔 이런 생각을 한다.

프로그래밍도, 소프트웨어도, 웹사이트도 결국은  
**현실을 옮겨 적은 것에 불과한 것 아닐까.**

웹사이트는 가게를 닮았다.

- 장바구니는 쇼핑 카트를 닮았고  
- 결제 페이지는 계산대를 닮았고  
- 리뷰는 입소문을 닮았고  
- 프로필은 명함이나 얼굴을 닮았다  

처음엔 분명히 그랬다.

현실에 있던 것을 인터넷 위로 옮겨놓는 일.  
불편한 동선을 줄이고, 시간을 압축하고, 거리를 없애는 일.

말하자면 **디지털은 현실의 축소판**이었다.  
소프트웨어는 현실을 조금 더 편하게 만든 복제품이었다.

---

# AI도 결국 모방에서 시작한다

AI도 비슷하다.

- 말을 따라 하고  
- 문장을 흉내 내고  
- 생각하는 척하고  
- 위로하는 법, 설명하는 법, 설득하는 법까지 배운다  

처음에는 이것도 단지 모방처럼 보였다.

사람이 쓰는 언어를 배워서  
사람처럼 말하는 기계.

그러니까 결국 **인간의 그림자** 같은 존재였다.

---

# 그런데 모방이 충분히 정교해지면

무서운 건 **모방이 계속 정교해질 때**다.

처음에는 모방이 항상 원본보다 아래에 있었다.

- 그림은 풍경을 대신하지 못했고  
- 지도는 도시를 대신하지 못했고  
- 온라인 쇼핑몰은 시장 골목의 공기까지 옮기지 못했다  

하지만 모방이 충분히 편리해지고,  
충분히 빠르고,  
충분히 익숙해지면

어느 순간 사람은 **원본보다 모방 쪽에 더 오래 머물기 시작한다.**

---

# 웹사이트는 가게의 복제품이었지만

웹사이트는 원래 **가게를 닮은 복제품**이었다.

하지만 지금은 조금 다르다.

사람들은 물건을 사기 위해 거리로 나가기보다  
검색창에 단어를 입력한다.

손에 쥐어보고 망설이기보다  
후기와 별점을 먼저 본다.

그러다 보니 현실의 가게도 바뀌기 시작한다.

- 온라인처럼 빠르게 응대해야 하고  
- 온라인처럼 편리해야 하고  
- 온라인처럼 설명 가능해야 한다  

즉

```
웹사이트 → 가게의 모방
가게 → 웹사이트의 기준
```

이 구조가 된다.

---

# AI도 같은 방향으로 가고 있을지 모른다

지금은 아직

```
사람 → 기준
AI → 모방
```

처럼 보인다.

AI는

- 사람의 말투를 배우고  
- 사람의 판단을 닮고  
- 사람의 감정을 설명하려 한다  

하지만 어느 순간 질문이 바뀐다.

```
이게 사람을 닮았는가?
```

에서

```
이게 충분히 쓸 만한가?
```

로.

그 순간 기준도 바뀐다.

---

# 모방이 기준이 되는 순간

예전에는 사람이 기준이었다.

하지만 언젠가 이렇게 될 수도 있다.

사람도

- AI처럼 빠르게 답해야 하고  
- AI처럼 정리해야 하고  
- AI처럼 친절해야 하고  
- AI처럼 쉬지 않고 반응해야 하는 세상  

그게 조금 무섭다.

모방이 무서운 이유는  
가짜가 진짜처럼 보여서가 아니다.

더 무서운 건

> **진짜가 점점 가짜의 속도를 따라가야 하는 상황**

이다.

---

# 인간은 원래 비효율적이다

사람은 원래 느리다.

- 생각은 끊기고  
- 감정은 흔들리고  
- 판단은 늦고  
- 말은 서툴다  

그런데 사람을 닮은 기계가  
너무 매끈하게 말하기 시작하면

우리는 점점

```
서툰 인간
<
매끈한 모방
```

을 더 선호하게 될지도 모른다.

---

# 현실이 복제품을 따라가기 시작하면

웹사이트가 가게를 모방했는데  
이제 가게가 웹사이트를 닮아가고,

AI가 사람을 모방했는데  
이제 사람이 AI처럼 일하기를 요구받는다면

그 다음에는 무엇이 남을까.

아마 점점 더 많은 것들이

```
현실 → 모방
모방 → 기준
```

이 되는 방향으로 갈 것이다.

---

# 사라지는 것들

그 과정에서 사라지는 것들은  
대개 이렇게 불린다.

```
비효율
```

예를 들어

- 느린 망설임  
- 말로 설명 안 되는 감정  
- 쓸모없어 보이는 침묵  
- 정리되지 않는 생각  
- 비논리적인 애정  

하지만 사실 인간은  
원래 그런 것들로 이루어져 있다.

---

# 그래서 무섭다

기술이 무서운 이유는  
강해져서가 아니다.

무서운 건  
**우리가 점점 그 기술이 이해하기 쉬운 형태로 변할 때**다.

웹사이트가 가게를 모방한 것이 아니라  
가게가 웹사이트를 닮아가고,

AI가 사람을 모방한 것이 아니라  
사람이 AI처럼 되기를 요구받는 세상.

---

# 마지막 생각

기술의 질문은 보통 이것이다.

```
얼마나 더 잘 만들 수 있는가
```

하지만 지금 우리가 물어야 할 질문은  
어쩌면 이것일지도 모른다.

> **무엇만큼은 끝까지 모방되지 않아야 하는가.**

느림, 망설임, 서툼, 진짜 마음 같은 것들.

왜냐하면 현실은 복제될 수 있어도  
**살아 있는 감각까지 완전히 복제되지는 않기 때문**이다.

그리고 어쩌면  
우리가 끝까지 인간으로 남는 이유도  
바로 그 복제되지 않는 부분들 때문일 것이다.
