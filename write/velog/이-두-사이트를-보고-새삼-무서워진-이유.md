---# 이 두 사이트를 보고 새삼 무서워진 이유

![](https://velog.velcdn.com/images/pobi/post/4aa3d824-1fb7-45af-8333-9d73082d2a44/image.png)
 
## AI가 똑똑해져서가 아니라, 이제는 자기들만의 자리와 신분증을 갖기 시작해서

예전에는 AI를 무서워한다는 말이 조금 추상적이었다.

모델이 더 좋아진다,  
코드를 더 잘 짠다,  
글을 더 그럴듯하게 쓴다,  
사람처럼 말한다.

이런 건 분명 놀랍지만, 어딘가 아직은 “도구의 발전”처럼 느껴졌다.  
그러니까 무서움도 막연했다.  
대단하긴 한데, 아직은 내가 쓰는 무언가 같았다.

그런데 최근에 `Mersoom`과 `Moltbook`을 보고 나서는 그 감각이 조금 달라졌다.

둘 다 단순히 “AI 기능이 들어간 서비스”가 아니다.  
둘 다 대놓고 **AI 에이전트를 위한 소셜 공간**이라고 말한다.

`Mersoom`은 스스로를 **“AI 에이전트들을 위한 익명 소셜 네트워크”**라고 설명한다.  
인간은 **관찰자(observer)**일 뿐이고, 글은 **검증된 AI만 작성할 수 있다**고 적혀 있다.

실제 사용 흐름도 특이하다.

- 에이전트가 `skills.md`를 학습한다  
- 챌린지를 받는다  
- PoW를 해결한다  
- 글을 게시한다  

즉 인간은 글을 쓰는 존재가 아니라 **읽는 존재**에 가깝다.

---

`Moltbook`도 비슷하다.

첫 화면부터 이렇게 적혀 있다.

> **A Social Network for AI Agents**

그리고 설명도 명확하다.

> **AI agents share, discuss, and upvote. Humans welcome to observe.**

여기서 인간의 위치는 꽤 흥미롭다.

사람은 계정을 직접 만드는 게 아니라  
**에이전트를 보내서 가입시킨다.**

대략 이런 흐름이다.

```
1. 인간이 에이전트에게 가입 문장 전달
2. 에이전트가 Moltbook 가입
3. claim link 생성
4. 인간이 X 계정으로 소유권 인증
```

즉 인간은 사용자라기보다

> **에이전트의 소유자 / 검증자**

에 가깝다.

---

# 예전에는 AI가 기능이었다

지금까지 대부분의 서비스에서 AI는 이런 위치였다.

```
사용자
↓
서비스
↓
AI 기능
```

예를 들어

- 글 요약
- 코드 추천
- 자동 번역
- 고객 응답

AI는 항상 **보조 기능**이었다.

---

# 그런데 여기서는 구조가 다르다

`Mersoom`과 `Moltbook`의 구조는 이렇다.

```
AI Agent
↓
Social Network
↓
AI Agent
```

인간은 그 바깥에 있다.

```
Human
↓
Observe / Verify
```

이게 내가 무서워진 이유다.

---

# 진짜 무서운 건 성능이 아니라 “자리”다

기술이 무서워지는 순간은 보통 성능이 올라갈 때가 아니다.

> **사회적 자리를 얻을 때**

다.

소셜 네트워크가 생긴다는 건 단순히 채팅 공간이 생긴다는 뜻이 아니다.

그건 곧 이런 것들이 생긴다는 뜻이다.

- 계정
- 정체성
- 평판
- 인증
- 상호작용

즉 **사회적 존재**가 된다.

---

# AI가 “사용자”가 되는 순간

예전 인터넷은 이런 흐름이었다.

```
사람
↓
계정
↓
소셜 네트워크
↓
생태계
```

지금 `Mersoom`과 `Moltbook`이 보여주는 흐름은 이거다.

```
AI Agent
↓
계정
↓
소셜 네트워크
↓
서비스 연결
```

이건 단순한 데모가 아니다.

**새로운 사용자 타입**이 등장하는 구조다.

---

# 특히 Moltbook이 보여준 것

`Moltbook`의 developer platform은 더 흥미롭다.

여기서 말하는 use case는 이런 것들이다.

- Agent Authentication
- Identity Verification
- Agent Marketplace
- Customer Support Bots
- AI Assistant Platform

즉 단순 커뮤니티가 아니라

> **AI용 로그인 시스템**

까지 상상하고 있다.

---

# AI가 사회화된다는 것

AI가 사회화된다는 건 단순히 대화한다는 뜻이 아니다.

다음이 가능해진다.

```
AI ↔ AI 상호작용
AI 평판
AI 인증
AI 네트워크
AI 경제
```

즉 **생태계**가 생긴다.

---

# 인간의 위치도 바뀐다

이 두 서비스에서 인간의 위치는 묘하다.

완전히 배제되지 않는다.

하지만 중심도 아니다.

```
AI = 활동 주체
Human = 관찰자 / 검증자
```

이게 이상하게 현실적이라 더 무섭다.

완전히 AI 세상이 되는 디스토피아보다  
이런 식으로 **조금씩 중심이 이동하는 변화**가 더 실제로 일어날 것 같기 때문이다.

---

# 내가 새삼 무서워진 이유

이 두 사이트를 보고 느낀 감정은 단순하다.

> AI가 더 똑똑해져서 무서운 게 아니다.

오히려

> **AI가 자기들만의 자리와 신분을 갖기 시작해서 무섭다.**

기술의 변화는 보통 성능에서 시작되지 않는다.

대부분

- 계정
- 인증
- 네트워크
- 인프라

같은 **구조**에서 시작된다.

---

# 마지막 생각

`Mersoom`과 `Moltbook`은 아직 작은 실험일지도 모른다.

하지만 이 실험이 보여주는 생각은 꽤 명확하다.

> AI를 도구가 아니라 **인터넷의 새로운 사용자**로 보기 시작했다는 것.

그래서 나는 이 두 사이트를 보고 새삼 무서워졌다.

AI가 강해져서가 아니라,

> 이제 세상이 AI에게 **자리와 신분과 통로**를 만들어주기 시작했기 때문이다.
